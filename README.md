
Implementirano je:

- pocetna forma koja zahtijeva unos imena ili nadimka da bi se igrica zapocela
- forma sa 3 nivoa koje korisnik može da bira
- forma gdje igrica pocinje
- info polje koje otvara formu gdje pišu detalji o pravilima igre
- dark/light mode
- pomocno slovo
- dugme koje otvara formu gdje se prikaziva korisnikovo trenutni uspjeh
- dugme za sljedecu rijec
- numerisanje koje ide nakon svake nove rijeci, tipa: 3/20
- bodovanje svake pogodene rijeci
- dvije nove opcije koje korisnik može otvoriti ako ostvari dovoljan broj bodova
- korisnik može unositi ili jedno slovo ili cijelu rijec u polje za unos
- korisnik je ogranicen na broj dozvoljenih puta da pogriješi (specificno za svaki level)
- ako korisnik nije pogodio rijec a ispunio je broj dozvoljenih pokušaja, tražena rijec se pojavi na ekranu
- dugme Nazad koje vraca korisnika na pogadanje rijeci iz pocetne faze
- svaka pogodena ili nepogodena rijec se spašava u listu, te se više ne pojavljuje korisniku
- rijeci se biraju random metodom
